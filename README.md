# Jetson-UART-Communication
UART Communication using STM32H753ZI (NUCLEO-H753ZI)  This project demonstrates UART (USART1) serial communication on the STM32H753ZI NUCLEO development board using STM32CubeIDE and STM32CubeMX. The firmware initializes the STM32H7 peripherals and transmits data over the Virtual COM Port (VCP) for communication with a host PC via a serial terminal.
# UART Communication using STM32H753ZI

This project demonstrates basic UART communication on the STM32H753ZI NUCLEO development board using STM32CubeIDE and the STM32 HAL library.

## Features

- UART (USART1) communication
- STM32 HAL drivers
- Developed using STM32CubeIDE
- Serial output via ST-LINK Virtual COM Port

## Hardware

- STM32H753ZI NUCLEO-144 Board
- USB Cable

## Software

- STM32CubeIDE
- STM32CubeMX
- Serial Terminal (PuTTY / Tera Term)
